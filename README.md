# SciCalculator
A Python implementation of a scientific calculator
![thumbnail] (./thumbnail.png)

## Dependencies
[Python Lex-Yacc](https://github.com/dabeaz/ply)  
[PyQt4](https://wiki.python.org/moin/PyQt4)  

## Features
** Arithmetic: **
- [x] Basic arithmetic
- [x] Fraction arithmetic
- [x] Complex number arithmetic

** Conversion:**
- [x] decimals <=> friction
- [x] degree <=> radian
- [x] polar <=> rectangular (for complex number)

** Built-in Functions:**
- [x] exponential
- [x] logarithmic
- [x] trigonometric
- [x] combinatorial

** Polynomial:**
- [x] addition
- [x] multiplication
- [x] derivative
- [x] integral
- [x] interpolation
- [x] least square fitting

** Vector:**
- [x] linear operation
- [x] inner product
- [x] cross product
